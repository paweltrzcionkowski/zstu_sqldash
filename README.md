**ZSTU_sqlDash** to lekka, webowa aplikacja do zarządzania bazami danych SQL, inspirowana phpMyAdminem. Stworzona po to, żeby ułatwić życie i pozwolić na zarządzanie tabelami z poziomu przeglądarki – szybko, sprawnie i bez konieczności klepania wszystkiego z palca w terminalu.

## 🔥 Główne Funkcje
* **Wygodny Dashboard:** Szybki podgląd bazy danych, relacji i struktury tabel.
* **Pełny CRUD:** Przeglądaj, dodawaj, edytuj i usuwaj rekordy za pomocą prostego interfejsu.
* **Konsola SQL:** Miejsce na Twoje własne, niestandardowe zapytania.
* **Zarządzanie strukturą:** Tworzenie i usuwanie tabel bez pamiętania na pamięć składni `ALTER TABLE`.
* **Zero zamulania:** Lekki kod, który działa natychmiast – bez setek zbędnych skryptów w tle.

## 🛠️ Tech Stack
* **Backend:** PHP
* **Frontend:** HTML, CSS
* **Bazy danych:** MySQL / MariaDB

## 🚀 Jak tego użyć?

1. Pobierz kod projektu (kliknij zielony przycisk **Code** -> **Download ZIP** go rozpakuj).
2. Wrzuć pliki na swój serwer lokalny lub produkcyjny (np. do katalogu `var/www/html` lub `htdocs`).
3. Skonfiguruj połączenie z bazą w pliku konfiguracyjnym *(tutaj możesz dopisać nazwę pliku, np. config.php)*.
4. Odpal aplikację w przeglądarce i zarządzaj bazami jak człowiek.# zstu_sqldash
